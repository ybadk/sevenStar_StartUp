## 💾 How Data is Stored

### Database Schema

```json
{
  "repos": {
    "owner/repo": {
      "id": "owner/repo",
      "name": "owner/repo",
      "description": "...",
      "industry": "AI/ML",
      "stars": 5000,
      "app_stars": 45,
      "user_rating": 6.5,
      "github_url": "https://github.com/...",
      "created_at": "2025-04-03T...",
      "tags": [...]
    }
  },
  "video_uploads": [...],
  "video_requests": [...]
}
```

### File Storage

- **All data** persists in `data/database.json`
- **Videos** stored in `data/video_uploads/`
- **Backups** - Simply copy `data/` folder
- **No cloud** - Everything stays local

